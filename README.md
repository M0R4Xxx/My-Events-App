# My-Events-App
Platform manajemen acara untuk menyederhanakan koordinasi tamu dan pelacakan RSVP secara real-time. Kelola undangan dan verifikasi kehadiran dengan integrasi token yang aman. Dibangun dengan Next.js 15, Prisma ORM, dan Tailwind CSS, aplikasi ini menawarkan antarmuka yang responsif, modular, dan berperforma tinggi untuk kebutuhan event Anda.
