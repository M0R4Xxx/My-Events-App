# My Events App

Platform manajemen acara untuk menyederhanakan koordinasi tamu dan pelacakan RSVP secara real-time. Kelola undangan dan verifikasi kehadiran dengan integrasi token yang aman. Dibangun dengan Next.js 15, Prisma ORM, dan Tailwind CSS, aplikasi ini menawarkan antarmuka yang responsif, modular, dan berperforma tinggi.

## 🚀 Akses Aplikasi
- **Live Demo:** [Masukkan Link Vercel Anda di sini]

## 🛠️ Teknologi Utama
- **Framework:** Next.js 15 (App Router)
- **Database:** PostgreSQL (Neon) & Prisma ORM
- **Styling:** Tailwind CSS
- **Authentication:** JWT (JSON Web Token)
- **Storage:** UploadThing

## ⚙️ Setup Lokal
1. Clone repository: `git clone https://github.com/M0R4Xxx/My-Events-App.git`
2. Install dependencies: `npm install`
3. Salin `.env.example` ke `.env` dan isi variabelnya.
4. Push schema ke database: `npx prisma db push`
5. Jalankan aplikasi: `npm run dev`

## 🏗️ Arsitektur
Aplikasi ini menggunakan pola **Server Actions** dan struktur folder modular untuk memastikan kode yang bersih, efisien, dan mudah dipelihara.
